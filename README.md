# Final Project: Space Kitty Game

View the deployed template [here](https://hjchoi28.github.io/fp2-final/).

This is the final project for HCDE 438: Web Technologies. Space Kitty is a 2D game, where the player uses their mouse to guide and keep the space kitty from colliding with asteroids. I used the p5.js and p5.play libraries for the game, as well as firebase for the scoreboard.
